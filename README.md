
# API Demo

API de demostración, desarrollada con Laravel 7.4.

### Instalación
Se requiere:

 - PHP 7.4 
 - Mysql
 
  

>       $ git clone https://github.com/luciobian/api-colegio.git
>       $  cd <project folder>
>       $ composer install
>       $ php artisan key:generate

## Ejecutar migraciones:

    $ cd <project folder>
    $ php artisan migrate
        

## Cargar datos:

    $ cd <project folder>
    $ php artisan db:seed
